# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [1.0.0] - 2026-04-06

### Added
- `idea-validation` — Validate business ideas before building
- `brainstorm` — Divergent thinking and idea generation sessions
- `software-architect` — System design and architecture decisions
- `senior-backend` — Production-grade backend development
- `devops-deploy` — Deploy applications and set up infrastructure
- `seo` — Search engine optimization for startups
- `landing-page` — Build high-conversion landing pages
- `analytics` — Product analytics and metrics setup
- `branding` — Brand identity and visual guidelines
- `legal-compliance` — Legal document templates and compliance checklists
- Journey-based README ("From Zero to Revenue")
- CODE_OF_CONDUCT.md
- Issue templates (bug report, skill request, improvement)
- Pull request template

### Changed
- `dev-workflow` — Now orchestrates `software-architect`, `senior-frontend`, and `senior-backend`
- README rewritten with journey view + category tables
- CONTRIBUTING.md updated with new categories

### Removed
- `game-dev` — Out of scope
- `remotion` — Too niche
- `web-design-guidelines` — Superseded by `senior-frontend`
